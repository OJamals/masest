# MASEST CMS Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a staff-managed CMS foundation for non-commerce public content while preserving the existing Supabase-backed product catalog, quote-first commerce rules, static public rendering, and SEO generation.

**Architecture:** Build a repo-native `ContentRepository` module backed by Supabase tables, exposed through Cloudflare Pages Functions, edited inside the existing admin shell, and exported into static JSON snapshots for public pages. Keep product pricing, variants, inventory, checkout, and quote routing outside the generic CMS because those are commerce domain data with existing admin APIs and invariants.

**Tech Stack:** Cloudflare Pages Functions, Supabase Postgres/Auth/Storage, vanilla JS admin modules, Node build tools, static HTML/CSS/JS storefront, `node --test`, Playwright for visual/admin smoke checks.

---

## Context Snapshot

- The public site is a static HTML/CSS/JS storefront with shared chrome and Node build/verification tooling.
- Product catalog truth already lives in Supabase at runtime; `data/catalog.seed.json` is bootstrap/disaster reset input.
- Existing admin sections already cover products, pricing, orders, accounts, messages, quotes, emails, traffic, and SEO audit surfaces.
- Services are still exported as public JSON through `data/services.json` and generated by `tools/build-catalog.mjs`.
- SEO/product-page generation is centralized in `tools/seo-inject.mjs`.
- The first CMS slice should avoid checkout and product-buying logic.

## CMS Ownership Boundary

CMS owns:

- Services and service packages display fields.
- Proof/case-study cards and metadata.
- Resource cards, FAQ blocks, and page supporting content.
- Industry/use-case cards.
- Page-level SEO fields where they can safely feed `tools/seo-inject.mjs`.
- Content asset metadata: storage path, alt text, dimensions, focal point, and usage.

CMS does not own:

- Product `mode`, price, variant, stock, freight, hazmat, tax, Stripe, QBO, order, company, or quote pipeline data.
- The product buy-vs-quote contract.
- Generated static product pages except through approved SEO/display fields.

## External CMS Position

Recommended first move: build the internal Supabase-backed CMS module.

Reasoning:

- It reuses the existing auth, staff role, admin UI, Cloudflare Functions, Supabase schema, and verification gates.
- It avoids splitting content truth across a hosted editorial system before the local domain boundaries are settled.
- It can still expose a clean adapter seam later if Directus or another database-first admin is worth adding.

Hold external CMS adoption until one of these becomes true:

- Non-technical editors need a richer editing surface than the in-house admin can reasonably provide.
- Multi-locale, scheduled publishing, approval workflows, or visual editing become near-term requirements.
- The team wants a dedicated content product and accepts the extra hosting, auth, webhook, preview, and billing surface.

Best later-fit external option: Directus, because it can sit over SQL-style content collections. Sanity or Contentful fit pure editorial workflows better, but they would split MASEST content from the current Supabase commerce/admin source of truth. Strapi adds a full self-hosted app and should be considered only if the project intentionally adopts that operational burden.

```sql
-- MASEST content CMS foundation
-- Staff writes through service-role Cloudflare Functions. Public pages consume generated snapshots.

create extension if not exists pgcrypto;

do $$ begin
  create type content_status as enum ('draft','published','archived');
exception when duplicate_object then null; end $$;

create table if not exists public.content_entries (
  id uuid primary key default gen_random_uuid(),
  type text not null,
  slug text not null,
  title text not null,
  status content_status not null default 'draft',
  locale text not null default 'en',
  payload jsonb not null,
  seo jsonb not null default '{}'::jsonb,
  version int not null default 1,
  published_at timestamptz,
  created_by uuid references auth.users(id) on delete set null,
  updated_by uuid references auth.users(id) on delete set null,
  created_at timestamptz not null default now(),
  updated_at timestamptz not null default now(),
  unique (type, slug, locale)
);

create index if not exists content_entries_lookup_idx
  on public.content_entries (type, status, locale, slug);

create table if not exists public.content_revisions (
  id uuid primary key default gen_random_uuid(),
  entry_id uuid not null references public.content_entries(id) on delete cascade,
  version int not null,
  status content_status not null,
  payload jsonb not null,
  seo jsonb not null default '{}'::jsonb,
  note text,
  author_id uuid references auth.users(id) on delete set null,
  created_at timestamptz not null default now(),
  unique (entry_id, version)
);

create table if not exists public.content_assets (
  id uuid primary key default gen_random_uuid(),
  storage_path text not null unique,
  alt text not null,
  mime_type text,
  width int,
  height int,
  focal_point jsonb not null default '{}'::jsonb,
  usage jsonb not null default '[]'::jsonb,
  created_by uuid references auth.users(id) on delete set null,
  created_at timestamptz not null default now()
);

alter table public.content_entries enable row level security;
alter table public.content_revisions enable row level security;
alter table public.content_assets enable row level security;

grant usage on schema public to service_role;
grant select, insert, update, delete on public.content_entries to service_role;
grant select, insert, update, delete on public.content_revisions to service_role;
grant select, insert, update, delete on public.content_assets to service_role;
```

```js
export const CONTENT_TYPES = new Set([
  'service',
  'service_package',
  'proof_card',
  'resource_card',
  'industry_card',
  'faq_block',
  'page_meta',
]);

export const CONTENT_STATUSES = new Set(['draft', 'published', 'archived']);

export function normalizeSlug(value) {
  return String(value || '')
    .trim()
    .toLowerCase()
    .replace(/[^a-z0-9]+/g, '-')
    .replace(/^-+|-+$/g, '');
}

export function normalizeContentEntry(input = {}) {
  const type = String(input.type || '').trim();
  const status = String(input.status || 'draft').trim();
  const title = String(input.title || '').trim();
  const slug = normalizeSlug(input.slug || title);
  return {
    id: input.id || undefined,
    type,
    slug,
    title,
    status,
    locale: String(input.locale || 'en').trim() || 'en',
    payload: input.payload && typeof input.payload === 'object' ? input.payload : {},
    seo: input.seo && typeof input.seo === 'object' ? input.seo : {},
    version: Number.isFinite(Number(input.version)) ? Number(input.version) : 1,
  };
}

export function validateContentEntry(input = {}) {
  const entry = normalizeContentEntry(input);
  if (!CONTENT_TYPES.has(entry.type)) return { ok: false, error: `Unsupported content type: ${entry.type}` };
  if (!CONTENT_STATUSES.has(entry.status)) return { ok: false, error: `Unsupported content status: ${entry.status}` };
  if (!entry.slug) return { ok: false, error: 'slug_required' };
  if (!entry.title) return { ok: false, error: 'title_required' };
  if (!entry.payload || typeof entry.payload !== 'object' || Array.isArray(entry.payload)) {
    return { ok: false, error: 'payload_object_required' };
  }
  return { ok: true, entry };
}

export function publicContentSnapshot(entries = []) {
  return entries
    .filter((entry) => entry.status === 'published')
    .reduce((acc, entry) => {
      const bucket = acc[entry.type] || [];
      bucket.push({
        type: entry.type,
        slug: entry.slug,
        title: entry.title,
        locale: entry.locale,
        payload: entry.payload,
        seo: entry.seo,
      });
      acc[entry.type] = bucket;
      return acc;
    }, {});
}

export function createContentRepository(sb) {
  return {
    async list({ type, status = 'published', locale = 'en' } = {}) {
      let query = sb.from('content_entries').select('*').eq('locale', locale);
      if (type) query = query.eq('type', type);
      if (status) query = query.eq('status', status);
      const { data, error } = await query.order('updated_at', { ascending: false });
      if (error) throw error;
      return data || [];
    },

    async saveDraft(input, userId) {
      const validation = validateContentEntry({ ...input, status: 'draft' });
      if (!validation.ok) return validation;
      const row = {
        ...validation.entry,
        updated_by: userId || null,
        updated_at: new Date().toISOString(),
      };
      const { data, error } = await sb
        .from('content_entries')
        .upsert(row, { onConflict: 'type,slug,locale' })
        .select('*')
        .single();
      if (error) throw error;
      return { ok: true, entry: data };
    },

    async publish(input, userId) {
      const validation = validateContentEntry({ ...input, status: 'published' });
      if (!validation.ok) return validation;
      const row = {
        ...validation.entry,
        status: 'published',
        published_at: new Date().toISOString(),
        updated_by: userId || null,
        updated_at: new Date().toISOString(),
      };
      const { data, error } = await sb
        .from('content_entries')
        .upsert(row, { onConflict: 'type,slug,locale' })
        .select('*')
        .single();
      if (error) throw error;
      return { ok: true, entry: data };
    },

    async archive({ type, slug, locale = 'en' }, userId) {
      const { data, error } = await sb
        .from('content_entries')
        .update({ status: 'archived', updated_by: userId || null, updated_at: new Date().toISOString() })
        .eq('type', type)
        .eq('slug', slug)
        .eq('locale', locale)
        .select('*')
        .single();
      if (error) throw error;
      return { ok: true, entry: data };
    },
  };
}
```

```js
import { adminClient } from '../../_lib/supabase.js';
import { json, readBody } from '../../_lib/http.js';
import { requireStaff, staffCan } from '../../_lib/staff.js';
import { createContentRepository } from '../../_lib/content.js';

export async function onRequest({ request, env }) {
  const { user, staff, role } = await requireStaff(request, env);
  if (!user) return json(401, { error: 'unauthenticated' });
  if (!staff) return json(403, { error: 'forbidden' });

  const repo = createContentRepository(adminClient(env));

  if (request.method === 'GET') {
    const url = new URL(request.url);
    const type = url.searchParams.get('type') || undefined;
    const status = url.searchParams.get('status') || 'published';
    const locale = url.searchParams.get('locale') || 'en';
    try {
      const entries = await repo.list({ type, status, locale });
      return json(200, { entries });
    } catch (error) {
      return json(500, { error: error.message });
    }
  }

  if (request.method === 'POST') {
    if (!staffCan(role, 'content.write')) {
      return json(403, { error: 'forbidden', message: 'Editing content requires owner access.' });
    }
    const body = await readBody(request);
    try {
      const result = body.publish
        ? await repo.publish(body.entry || body, user.id)
        : await repo.saveDraft(body.entry || body, user.id);
      if (!result.ok) return json(400, { error: result.error });
      return json(200, result);
    } catch (error) {
      return json(500, { error: error.message });
    }
  }

  if (request.method === 'DELETE') {
    if (!staffCan(role, 'content.write')) {
      return json(403, { error: 'forbidden', message: 'Archiving content requires owner access.' });
    }
    const body = await readBody(request);
    try {
      const result = await repo.archive(body, user.id);
      return json(200, result);
    } catch (error) {
      return json(500, { error: error.message });
    }
  }

  return json(405, { error: 'method_not_allowed' });
}
```

```js
const TYPES = [
  ['service', 'Services'],
  ['service_package', 'Service packages'],
  ['proof_card', 'Proof cards'],
  ['resource_card', 'Resource cards'],
  ['industry_card', 'Industry cards'],
  ['faq_block', 'FAQ blocks'],
  ['page_meta', 'Page metadata'],
];

function typeOptions(selected = 'service') {
  return TYPES.map(([value, label]) => `<option value="${value}"${value === selected ? ' selected' : ''}>${label}</option>`).join('');
}

function entryForm(entry = {}) {
  const payload = JSON.stringify(entry.payload || {}, null, 2);
  const seo = JSON.stringify(entry.seo || {}, null, 2);
  return `
    <div class="adm-card">
      <div class="adm-panel-header">
        <h2>Content editor</h2>
        <span class="badge" data-s="${entry.status || 'draft'}">${entry.status || 'draft'}</span>
      </div>
      <form id="contentForm" class="adm-form-grid" onsubmit="return false">
        <label>Type <select id="contentType" class="adm-select">${typeOptions(entry.type)}</select></label>
        <label class="wide">Title <input id="contentTitle" class="adm-input" value="${entry.title || ''}" required></label>
        <label class="wide">Slug <input id="contentSlug" class="adm-input" value="${entry.slug || ''}" required></label>
        <label class="full">Payload JSON <textarea id="contentPayload" class="adm-textarea">${payload}</textarea></label>
        <label class="full">SEO JSON <textarea id="contentSeo" class="adm-textarea">${seo}</textarea></label>
        <button id="contentSaveDraft" class="btn btn-secondary" type="button">Save draft</button>
        <button id="contentPublish" class="btn btn-primary" type="button">Publish</button>
      </form>
      <p id="contentStatus" class="adm-status" role="status" aria-live="polite"></p>
    </div>
  `;
}

function entryList(entries = []) {
  if (!entries.length) return '<div class="adm-card"><p class="muted">No entries found.</p></div>';
  return `
    <div class="adm-card">
      <h2>Published content</h2>
      <table class="adm">
        <thead><tr><th>Type</th><th>Title</th><th>Slug</th><th>Status</th></tr></thead>
        <tbody>
          ${entries.map((entry) => `
            <tr>
              <td>${entry.type}</td>
              <td>${entry.title}</td>
              <td>${entry.slug}</td>
              <td><span class="badge" data-s="${entry.status}">${entry.status}</span></td>
            </tr>
          `).join('')}
        </tbody>
      </table>
    </div>
  `;
}

export async function createContentTab({ api }) {
  const root = document.getElementById('admContent');
  if (!root) return;
  root.innerHTML = `${entryForm()}<div id="contentList" class="adm-table-wrap">Loading...</div>`;

  async function load() {
    const data = await api('/api/admin/content?status=published');
    document.getElementById('contentList').outerHTML = `<div id="contentList">${entryList(data.entries || [])}</div>`;
  }

  async function save(publish = false) {
    const status = document.getElementById('contentStatus');
    let payload;
    let seo;
    try {
      payload = JSON.parse(document.getElementById('contentPayload').value || '{}');
      seo = JSON.parse(document.getElementById('contentSeo').value || '{}');
    } catch (error) {
      status.dataset.state = 'err';
      status.textContent = `Invalid JSON: ${error.message}`;
      return;
    }
    const body = {
      publish,
      entry: {
        type: document.getElementById('contentType').value,
        title: document.getElementById('contentTitle').value,
        slug: document.getElementById('contentSlug').value,
        payload,
        seo,
      },
    };
    const result = await api('/api/admin/content', { method: 'POST', body: JSON.stringify(body) });
    status.dataset.state = result.ok ? 'ok' : 'err';
    status.textContent = result.ok ? 'Saved.' : (result.error || 'Save failed.');
    if (result.ok) await load();
  }

  document.getElementById('contentSaveDraft')?.addEventListener('click', () => save(false));
  document.getElementById('contentPublish')?.addEventListener('click', () => save(true));
  await load();
}
```

```js
import { mkdirSync, writeFileSync } from 'node:fs';
import { dirname, join } from 'node:path';
import { fileURLToPath } from 'node:url';
import { publicContentSnapshot } from '../functions/_lib/content.js';

const ROOT = dirname(dirname(fileURLToPath(import.meta.url)));
const OUT_DIR = join(ROOT, 'data/content');

function writeJson(path, value) {
  writeFileSync(path, `${JSON.stringify(value, null, 2)}\n`);
}

function servicesPayload(snapshot) {
  return {
    services: (snapshot.service || []).map((entry) => ({ slug: entry.slug, ...entry.payload, seo: entry.seo })),
    service_packages: (snapshot.service_package || []).map((entry) => ({ slug: entry.slug, ...entry.payload, seo: entry.seo })),
  };
}

async function main() {
  mkdirSync(OUT_DIR, { recursive: true });

  const source = process.env.CONTENT_EXPORT_SOURCE;
  if (!source) {
    writeJson(join(OUT_DIR, 'services.json'), { services: [], service_packages: [] });
    return;
  }

  const entries = JSON.parse(source);
  const snapshot = publicContentSnapshot(entries);
  writeJson(join(OUT_DIR, 'services.json'), servicesPayload(snapshot));
}

main().catch((error) => {
  console.error(error);
  process.exitCode = 1;
});
```
